# KinBridge Client Build Config

## `SERVER_PUBLIC_KEY.txt`
Ed25519 public key of the KinBridge server at 192.168.68.54. Generated by `hbbs` on first boot on 2026-04-18. Baked into the Android APK at build time so clients trust only Wilson's server.

**Counterpart private key:** `/var/lib/kinbridge/data/id_ed25519` on the Olares server. **Never commit.** Back it up before a server rebuild.

## How it's used
At Phase 3 build time, the APK build pipeline will:
1. Read this file
2. Pass it to the Flutter build as `--dart-define=CUSTOM_KEY=<contents>`
3. Pair it with `--dart-define=CUSTOM_RENDEZVOUS_SERVER=192.168.68.54`

## Rotate
If the server private key is lost or regenerated:
1. New `id_ed25519.pub` on the server → paste into this file.
2. Rebuild the APK.
3. Push update to all paired clients (they can't connect until updated).
